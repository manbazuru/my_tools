flowup.js

スクロールした時にいい感じにコンテンツがついてくるプラグイン

  • 参考URL : http://coliss.com/articles/build-websites/operation/javascript/jquery-plugin-flow-up.html
  • 公式URL：http://www.dominikgorecki.com/download/flow-up/

使い方

$(document).ready(function() {
    $("CONTAINER-SELECTOR").flowUp("FLOW-UP-ELEMENTS-SELECTOR");
 });

これだけ。

オプションで、遅れる時のY軸やタイミングが設定が可能。

$(document).ready(function() {
    $("CONTAINER-SELECTOR").flowUp("FLOW-UP-ELEMENTS-SELECTOR", { transalteY: 350, duration: 1 });
 });
